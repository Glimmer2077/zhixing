# 2026-06-29 - M2 Start

## Goal

Render the seed tree as a read-only card grid with navigation.

## Scope

- `Mark` SVG renderer for the M1 mark kinds.
- `Card` and `CardGrid` using the M1 palette, size, masonry, and date helpers.
- `Header` with root wordmark, nested back affordance, and settings affordance placeholder.
- App shell over `createSeedTree()` with drill-in and back navigation.
- Empty state for leaf nodes.
- Component and e2e tests for read-only browsing.

## Out of Scope

- Add, edit, delete, reorder, reparent.
- Zustand store, IndexedDB persistence, undo/redo.
- Settings sheet, import/export UI.

## Testing Intent

Tests should assert user-visible behavior: cards render with title/subtitle/date/mark,
grid renders root seed data, tapping a card enters its child grid, back returns to root,
and leaf cards show the empty state.
